Тест по информатике на тему "Кодирование, с использованием недесятичных систем исчеслений"

1 Все 5-буквенные слова, составленные из букв М, И, Р, записаныв алфавитном порядке. Вот начало
1. ИИИМИ,
2. ИИИММ,
3. ИИИМР,
4. ИИИРИ,
......
Какое слово стоит на 123 месте списка?
2 Все 5-буквенные слова, составленные из букв А, Б, В, записаныв алфавитном порядке. Вот начало
1. ВВВВВ,
2. ВВВВА,
3. ВВВВБ,
4. ВВВАА,
......
Какое слово стоит на 201 месте списка?
3 Все 5-буквенные слова, составленные из букв Т, О, Н, записаныв алфавитном порядке. Вот начало
1. ТТТТТ,
2. ТТТТО,
3. ТТТТН,
4. ТТТОТ,
......
Сколько букв Т встречается в слове, стоящем на 101 месте от начала списка?
4 Все 5-буквенные слова, составленные из букв Х, О, Й, записаныв алфавитном порядке. Вот начало
1. ХХХХХ,
2. ХХХХЙ,
3. ХХХХО,
4. ХХХЙЙ,
......
Сколько букв Й встречается в слове, стоящем на 201 месте от начала списка?
5 Все 5-буквенные слова, составленные из букв М, О, Т, записаныв алфавитном порядке. Вот начало
1. ТТТТТ,
2. ТТТТО,
3. ТТТТМ,
4. ТТТОО,
......
Укажите номер в списке слова, которое первым начинается с буквы М?
6 Все 5-буквенные слова, составленные из букв Ш, Е, Л, записаныв алфавитном порядке. Вот начало
1. ШШШШШ,
2. ШШШШЕ,
3. ШШШШЛ,
4. ШШШЕЕ,
......
Укажите номер в списке слова, которое первым начинается с буквы Е?
7 Строки (цепочки символов латинских букв) создаются по следующему правилу: - первая строка состоит из одного символа - цифры 1; каждая из последующих цепочек создаётся такими действиями: в очередную строку сначала записывается цифра, соответствующая номеру строки (на і-м шаге пишется цифра і), к ней слева дважды подряд приписывается предыдущая строка.
(1) 1,
(2) 112,
(3) 1121123,
(4) 112112311211234,
......
Запишите шесть цифр подряд, стоящих в седьмой строке со 116-го по 121-е место (считая слева направо).
8 Строки из латинских букв создаются по следующему правилу: - первая строка состоит из буквы «А»; в каждой из последующих строк сначала дважды подряд записывается предыдущая строка, а затем справа приписывается буква, порядковый номер которой в алфавите соответствует номеру строки (на і-м шаге пишется і-я буква алфавита).
(1) A,
(2) AAB,
(3) ААBAABC,
(4) AABAABCAABAABCD,
......
Латинский алфавит (для справки): ABCDEFGHIJKLMNOPQRSTUVWXYZ Запишите шесть символов подряд, которые записаны в восьмой строке с 100-го по 105- е место (считая слева направо).
